# Discover Kigali – Things To Do & Places To Eat

## Project Description
This project is a simple informational website created using **HTML and CSS**.  
It showcases popular **tourist attractions and places to eat around Kigali, Rwanda**.  
The website is designed to be user-friendly, well-structured, and visually appealing, helping visitors explore Kigali easily.

## HTML Pages Included
The project consists of the following HTML pages:

1. **index.html** – Home page linking to all other pages  
2. **mount-kigali.html** – Information about Mount Kigali and hiking activities  
3. **golf.html** – Details about Kigali Golf Resort & Villas  
4. **restaurants.html** – Popular restaurants and dining options in Kigali  
5. **(Any other page you included, if applicable)**  

Each page contains navigation links to move easily between pages.
